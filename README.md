Projeto - Cards

A página é uma apresentação breve de alguns personagens da série Hilda, utilizando uma formato de carta, 
cada uma possui o nome do respectivo personagem e um texto curto sobre o mesmo. 
Clicando nas setas ao lado da carta o usuário pode avançar ou voltar o personagem selecionado. São cartas
personagens no total.
